# Switch

For boolean values

## Options

| Option     | Description                  | Default |
| ---------- | ---------------------------- | ------- |
| `labelOn`  | Label when the switch is on  |         |
| `labelOff` | Label when the switch is off |         |
| `colorOn`  | Color when the switch is on  |         |
| `colorOff` | Color when the switch is off |         |
